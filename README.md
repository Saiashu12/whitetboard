Complete Setup Guide



### Clone the Project and move to whiteboard directory

```bash
git clone https://github.com/Saiashu12/whiteboard.git

cd whiteboard
```

### Backend (For External Deployment)
The backend folder contains a complete Node.js/Express API that you can deploy separately:

**Navigate to server directory:**
```bash
cd server
```

**Install dependencies:**
```bash
npm install
```

**Run in development:**
```bash
npm start
```

### Now run client
```bash
cd client
```
